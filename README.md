# CodeGuardian - AI代码信任验证平台

## 项目概述
CodeGuardian是一个开源的全栈AI代码信任与验证平台，解决2026年开发者面临的核心痛点：**96%的开发者不信任AI生成的代码，42%的代码已是AI写的但无人敢签字上线**。

## 核心痛点分析
1. **信任危机**：AI代码质量参差不齐，无人愿意承担上线风险
2. **上下文负债**：AI不了解业务历史、架构决策和隐知识
3. **幻觉问题**：AI生成看似合理但实际错误的代码
4. **审查瓶颈**：人工审查速度跟不上AI生成速度
5. **责任归属**：谁为AI代码的质量和安全性负责？

## 项目愿景
成为AI编程时代的"代码质量签名系统"，让开发者能够**信任、验证、并安全地部署AI生成的代码**。

## 核心功能
1. **多模型共识审查**：同时调用多个LLMs进行代码审查，形成共识
2. **AI代码信任评分(0-100)**：综合评分系统，量化代码可信度
3. **上下文负债检测**：识别代码中的隐知识缺口和架构不一致
4. **幻觉检测引擎**：检测AI生成的虚假或错误代码片段
5. **安全漏洞扫描**：集成SAST工具，检测安全风险
6. **业务逻辑一致性验证**：验证代码是否符合业务规则
7. **CI/CD无缝集成**：GitHub Actions、GitLab CI等集成

## 技术架构
- **前端**：React + TypeScript + TailwindCSS
- **后端**：FastAPI + Python
- **数据库**：PostgreSQL + Redis
- **AI模型**：OpenRouter API集成（Claude、GPT、DeepSeek等）
- **部署**：Docker + Kubernetes

## 5W1H架构
- **What**：AI代码信任验证平台
- **Why**：解决AI代码信任危机，提高开发效率和质量
- **Who**：开发者、团队负责人、安全工程师
- **When**：在代码提交、PR审查、CI/CD阶段
- **Where**：GitHub、GitLab、本地开发环境
- **How**：通过多模型共识、信任评分、上下文分析等技术
